# Type Test ⌨️

A simple and interactive typing speed test application built to improve typing accuracy and measure WPM (Words Per Minute) in real time.

## 🚀 Features

- Real-time typing speed calculation
- Accuracy tracking
- Dynamic random text generation
- Timer functionality
- Clean and responsive UI
- Instant performance feedback

## 🛠️ Tech Stack

- HTML
- CSS
- JavaScript

## ⚡ Getting Started

1. Clone the repository

```bash
git clone https://github.com/subxm/type-test.git
```

2. Open the project folder

```bash
cd type-test
```

3. Run the project

Simply open `index.html` in your browser.
